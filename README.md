## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Testing & Deployment on Vercel

Ready to share your portfolio with the world? Vercel makes deployment seamless and straightforward.

### Quick Deploy

1. **Push to GitHub** - Ensure your latest changes are committed and pushed to your repository
2. **Connect to Vercel** - Visit [vercel.com/new](https://vercel.com/new) and import your GitHub repository
3. **Configure & Deploy** - Vercel will automatically detect Next.js settings and deploy your app
4. **Live in Seconds** - Your portfolio will be live with a production URL instantly

### Features You Get

- ✅ **Automatic Deployments** - Every push to main triggers a new deployment
- ✅ **Preview Deployments** - Each pull request gets its own preview URL for testing
- ✅ **Custom Domains** - Add your own domain name for a professional touch
- ✅ **Analytics & Performance** - Monitor your site's performance and visitor insights
- ✅ **Edge Network** - Global CDN ensures fast loading times worldwide
